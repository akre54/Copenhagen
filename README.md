## Requirements
* RVM with 1.9.3-p327
* Postgres

## Setup
* Create your database.yml, `rake db:create`, `rake db:migrate` and `rake db:data:load db/data.yml`

## Running
* run zeus, pow, and whatever other rails betterments you have
* `rails s` or `zeus s` if ya got it
